ABOUT VADER:
--------------------------------------------------------------------------------
VADER (Valence Aware Dictionary and sEntiment Reasoner) is specifically
designed for social media text analysis. It:
  • Handles slang, emoticons, and informal language
  • Considers capitalization (EXCITED vs excited)
  • Recognizes punctuation emphasis (good!!! vs good)
  • Detects negations (not good)
  • Scores from -1 (most negative) to +1 (most positive)

CLASSIFICATION THRESHOLDS:
  • Positive: compound >= 0.05
  • Neutral: -0.05 < compound < 0.05
  • Negative: compound <= -0.05

================================================================================
BERT vs VADER COMPARISON
================================================================================

ABOUT BERT SENTIMENT:
--------------------------------------------------------------------------------
BERT (Bidirectional Encoder Representations from Transformers) is a
deep learning model that:
  • Understands context and word relationships
  • Captures nuanced meanings and sarcasm better
  • Requires more computational resources
  • Generally more accurate but slower

VADER vs BERT - Which is Better?
--------------------------------------------------------------------------------

VADER Advantages:
  ✓ Fast - can process millions of posts quickly
  ✓ No training data required
  ✓ Specifically designed for social media
  ✓ Handles emoticons, slang, and informal language
  ✓ Provides explainable scores (positive/negative/neutral components)
  ✓ Works well for medical/health discussions

BERT Advantages:
  ✓ Better at understanding context and nuance
  ✓ Captures complex sentence structures
  ✓ Can detect subtle sarcasm and irony
  ✓ Generally higher accuracy on complex texts

For This Analysis:
--------------------------------------------------------------------------------
VADER is likely more appropriate because:

1. Reddit posts contain informal language, abbreviations, emoticons
   → VADER is specifically designed for this

2. Large dataset (thousands of posts)
   → VADER's speed advantage is significant

3. Medical context with custom lexicon
   → We enhanced VADER with medical terminology adjustments

4. Need for explainability
   → VADER provides component scores (pos/neg/neu breakdown)

5. Real-time sentiment patterns
   → Fast processing enables temporal analysis

RECOMMENDATION:
  Use VADER for this analysis. If higher accuracy is needed for specific
  cases or research validation, BERT can be applied to a sample subset.
